## Solução de BI do PJE2

A solução de BI do PJE2 pode ser decomposta em dois módulos:

1. Módulo de BI: é o módulo que contém o ferramental e produtos voltados para a solução de BI.
    Dentre eles estão o Superset e Jasper Reports Server.

2. Módulo de software: esse é o módulo que contém a implementação de software para adaptação do PJE
    à utilização do BI. Mais especificamente, contém adaptações no PJE2-WEB e Microserviço Java para conexão com
    o módulo de BI.

### Instalação

Para instalar a solução, será necessário instalar-se os dois módulos mencionados acima.

* [Instalação do módulo de BI](./instalacao/modulobi.md)

* [Instalação do módulo de software](./instalacao/modulosoftware.md)
